# Sk4rd's Dotfiles

This repository contains all of my linux dotfiles.

### Install

Before installing, all your own config files will be backed up.

```
sudo ./install.sh
```

### Uninstall

This will remove all my config files, which have been applied, **forever**.

```
sudo ./uninstall.sh
```
